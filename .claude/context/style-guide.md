# cside Writing Style Guide

## General Principles
- Write at an 8th-10th grade reading level (Flesch-Kincaid)
- Average sentence length: 15-20 words
- Paragraphs: 2-4 sentences, one idea each
- 80%+ active voice
- Lead with the most important information

## Terminology Preferences

| Preferred | Avoid |
|-----------|-------|
| client-side security | web security (too broad) |
| third-party scripts | third-party code |
| e-skimming | eSkimming, e-Skimming |
| Magecart | magecart, MageCart |
| real-time | real time (as adjective) |
| cside | C/Side, CSide, C-Side |
| PCI DSS 4.0.1 | PCI-DSS, PCI 4.0 |
| checkout page | payment page (unless specifically about payment forms) |
| script monitoring | script scanning (monitoring implies continuous) |
| detect and block | find and stop |
| deploy | install (cside doesn't "install" on sites) |

## Formatting Standards

### Capitalization
- Sentence case for headings (not Title Case)
- Product names as branded: cside, PCI DSS, Magecart
- No ALL CAPS for emphasis

### Numbers
- Spell out one through nine, use numerals for 10+
- Always use numerals for: percentages, statistics, technical specs
- Use commas in numbers over 999: 1,000 not 1000

### Punctuation
- Oxford comma: "scripts, tags, and pixels"
- Em dashes: avoid (use commas, colons, or parentheses)
- Semicolons: use sparingly
- Exclamation marks: never in body copy, rarely in headings

### Lists
- Use bullet points for unordered items
- Use numbered lists for sequential steps
- Each item starts with a capital letter
- No trailing punctuation on list items unless they're full sentences
- Parallel structure across all items

## AI Writing Patterns to Avoid
- No em dashes (--) -- use commas, colons, or parentheses
- Avoid: delve, leverage, utilize, facilitate, comprehensive, robust, seamless, cutting-edge, game-changing
- No formulaic openings: "In today's digital landscape," "In an era of," "In the realm of"
- No hedging: "it's worth noting that," "it's important to understand"
- Vary sentence structure -- don't start consecutive sentences the same way
- Use specific language over vague generalities
